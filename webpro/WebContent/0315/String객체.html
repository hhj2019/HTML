indexOf("문자") : 문자가 몇번째에 있는지 검사

lastIndexOf("문자") : 문자를 뒤에서부터 찾아서 몇번째에 있는지 검사

subString(start, end) : start에서부터 end-1까지의 문자열을 추출

slice(start, end) : start에서부터 end-1까지의 문자열을 자름

substr(start, length) : start에서 시작하여 length 길이만큼 부분 문자열을 추출


split(구분자, 갯수) : 구분자를 기준으로 해서 문자열을 분리 한다 분리되는 문자의 갯수
